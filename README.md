## sensor-signal-processing
Analysis and noise reduction of simulated MEMS sensor data using Python

## Project Overview
This project focuses on analyzing and reducing noise in simulated MEMS sensor data using Python.

##  Methods Used
- Signal simulation using sine wave
- Addition of Gaussian noise
- Moving Average Filtering
- Butterworth Low-Pass Filtering
- Data visualization using Matplotlib

##  Results
The Butterworth filter effectively reduces high-frequency noise while preserving the original signal shape.

##  Tools & Technologies
- Python
- NumPy
- Matplotlib
- SciPy
